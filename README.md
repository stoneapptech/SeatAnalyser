# SeatAnalyser
An mini project that helps deciding the best method to switch seats.
So yeah, here's a simple explanation of that.

# Environment
Running this project requires the following packages installed with Python3:
1. Numpy
2. Matplotlib

# Method
* Insert (inst)  
```
Req: 5 f 2
Before: [1,2,3,4,5]
After: [1,2,5,3,4]
```

* Swap  
```
Req: 5 f 2
Before: [1,2,3,4,5]
After: [1,2,5,4,3]
```

* push  
```
Req: 5 f 2
Before: [1,2,3,4,5]
After: [3,4,5,1,2]
```
